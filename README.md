# 使用方法
### Utils
* Shortcut 快捷方法、属性
```
statusBarHeight：状态栏高度
windowSafeAreaInsets：安全区域
isNotchScreen: 是否刘海屏
isIPhoneX: 是否 iPhoneX
appDelegate: App代理
screenHeight: 屏幕高度
screenWidth: 屏幕宽度
```
* Constant 常用常亮
* UIImageUtil Image工具
```
dateImage: 图片左下角添加时间水印
gradiantImage: 渐变图片
cacheLocal: 缓存图片到本地
localImage: 读取缓存图片

```
* UIColorUtil Color工具
* SystemConfigure 系统配置
* UIFontUtil 字体工具
